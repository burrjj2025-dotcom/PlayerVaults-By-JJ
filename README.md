# PlayerVaults-By-JJ
Just another Player Vaults plugin for Paper 1.13 to 1.21.11. But it's made by me JJ. I tried many other versions of this plugin made by other people, and none of them did what I wanted. I wanted all players to have unlimited vaults with no LuckPerms required. And the plugin does not depend on anything but itself. You just install my plugin and done
